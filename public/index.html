How to Create the Cookies File (cookies.json)

Follow these steps to extract cookies from the Sienge Platform and save them as a cookies.json file. You only need Google Chrome and Notepad (available on any Windows computer).

  1. Log in to the Sienge Platform:
    • Open Google Chrome on your computer.
    • Go to https://npu2.sienge.com.br.
    • Log in with your credentials (e.g., click "Sign in with Sienge ID," select your email, and enter your password).
    • Ensure you're on the payments page where you see fields like "Data da Baixa."
  2. Open Chrome Developer Tools:
    • Press F12 on your keyboard, or right-click anywhere on the page and select Inspect.
    • A panel will open. Click the Application tab at the top (if you don't see it, click the double arrow >> and select Application).
  3. Find the Cookies:
    • In the left sidebar, expand the Cookies section by clicking the arrow next to it.
    • Click on https://npu2.sienge.com.br. A table will show all cookies, with columns like Name, Value, Domain, etc.
    • Right-click anywhere in the cookie table and select Copy all as JSON. This copies the cookies to your clipboard.
  4. Save Cookies to a File:
    • Open Notepad (search for "Notepad" in the Windows search bar).
    • Paste the copied content (Ctrl+V). It should start with [ and end with ].
    • Click File > Save As.
    • In the "File name" field, type cookies.json (include the .json extension).
    • In the "Save as type" dropdown, select All Files (*.*).
    • Choose a location (e.g., Desktop) and click Save.
  5. Upload the File:
    • Use the "Upload Cookies File" field above to select your cookies.json file.
    • Upload the Excel file and click "Start Automation" to run the process.
  6. Important Notes:
    • If the automation fails with a login error, your cookies may have expired. Repeat these steps to create a new cookies.json file.
    • Keep the cookies.json file secure, as it contains sensitive session information.
